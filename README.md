# Mars Collection App (CAPP)

**GitHub Repository:** [Mars Collection App](Link-https://github.com/thessocinco/mars-collection-app.git)

---

## Technology
- Java
- Android Studio
- SQLite

**Database Tool:** DB Browser for SQLite

**Application Type:** Field Local (Offline) Application

**Deployment Area:** Region 11 & 12

---

## Purpose
Manages small and large collection transactions and supports daily field operations

---

## Key Functions
- SMS-based data transmission for sending transaction data from the field to the central system
- GPS-enabled location tracking to capture and verify field transaction locations
- System message handling for operational alerts and updates
- Offline data storage using SQLite, with database management and inspection performed via DB Browser for SQLite

---

## Impact / Results
- Increased efficiency in collection and field transaction processes
- Improved timeliness and reliability of transaction reporting from field personnel
- Enhanced operational oversight and monitoring of collection activities
- Reliable offline data management with secure local storage and synchronization support
